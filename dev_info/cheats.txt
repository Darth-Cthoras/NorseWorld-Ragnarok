
Чит-коды игры
(регистр символов не имеет значения),
<> - условные символы, набирать не нужно.

set_divinity	- включение божественности: бессмертие, 
  любого, включая богов, убиваем с одного удара, даже без оружия, 
  "божественное перемещение" - кликами на карте и "око бога" - видение
  всех ловушек и прочей дряни...

takeitem <sign> <количество> - добавить предмет в рюкзак игрока
  <sign> = 'all_rings' - будут добавлены все существующие кольца,
  <sign> = 'all_potions' - будут добавлены все существующие снадобья,
  <sign> = 'all_wands' - будут добавлены все существующие палочки,
  <sign> = 'all_scrolls' - будут добавлены все существующие свитки
  <sign> = 'all_amulets' - будут добавлены все существующие амулеты.
  <sign> = 'all_armor' - будут добавлены все существующие доспехи и одежда.
  <sign> = 'all_foods' - будет добавлена вся существующая пища.
  <sign> = 'all_tools' - будут добавлены все существующие инструменты.
  <sign> = 'all_weapon' - будет добавлено все существующее оружие.
  <sign> = 'сигнатура' - будет добавлен предмет с заданной сигнатурой (подписью).

set_morality <значение> - установить мораль в заданное значение

kill_all - убить всех на текущем уровне

add_monster <sign> - добавить монстра на текущий уровень

transform <sign> - превратиться в заданного монстра

suicide - умереть на месте

show_goals - показать количество целей у каждой твари на уровне

set_freeze - "заморозить" всех тварей

set_fury - режим "атаковать всегда"
